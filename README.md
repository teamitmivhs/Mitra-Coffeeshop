# Kopi Mitra — Ngopi Asik di SMK Mitra Industri

Website resmi **Kopi Mitra**, coffee shop di lingkungan SMK Mitra Industri. Didesain dengan estetika Magazine Layout menggunakan palette warna Cream & Caramel.

---

### [STK] Tech Stack

Project ini dibangun dengan arsitektur modern untuk performa dan skalabilitas tinggi:

*   **Runtime:** Bun (v1.3.14)
*   **Framework:** TanStack Start
*   **Routing:** TanStack Router (Type-safe)
*   **Styling:** Tailwind CSS v4 (Engine: Lightning CSS)
*   **UI Primitives:** Radix UI & Shadcn UI
*   **Icons:** Lucide React

---

### [FTR] Fitur Utama

*   **Magazine-Style Design:** Implementasi layout asimetris yang responsif dan estetis.
*   **Menu Catalog:** Manajemen katalog produk kopi dan non-kopi.
*   **Student Promo System:** Integrasi banner promosi khusus civitas SMK Mitra Industri.
*   **Direct WhatsApp API:** Integrasi tombol pesan instan untuk pemesanan langsung.

---

### [OPS] Instruksi Operasional

Pastikan sistem Anda telah terkonfigurasi dengan **Bun**.

#### 1. Inisialisasi Repositori
```bash
git clone <url-repository>
cd mitra-coffeeshop
```

#### 2. Instalasi Dependensi
```bash
bun install
```

#### 3. Lingkungan Pengembangan
```bash
bun dev
```
Akses server lokal melalui: `http://localhost:3000`

#### 4. Kompilasi Produksi
```bash
bun build
```

---

### [DIR] Struktur Direktori

*   `src/routes/` : Struktur routing berbasis file (TanStack Router).
*   `src/components/` : Library komponen UI internal.
*   `src/assets/` : Manajemen aset statis (images & media).
*   `src/styles.css` : Definisi design system dan variabel Tailwind v4.
*   `remotion/` : Framework video programatis untuk konten promosi.
